Ayudas:
1. El navegador que esta utilizando debe soportar georreferenciación.
2. Asegurese de tener habilitado el permiso para la georreferenciación en el navegador.
3. Cada marcador en el mapa representa la ubicación donde fue capturada y analizada una fotografía.
4. Haga click sobre el marcador para observar la fotografía.
5. Mantenga el cursor ubicado sobre un marcador para conocer la dirección.
